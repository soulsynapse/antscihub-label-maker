# AntSciHub Label Maker

## Overview
This is a standardized label utility for avery 5267 style label sheets. It is meant to facilitate labels for insect colonies and scaling for computer vision.

I only put this under the antscihub umbrella so people can find it.

## Features
It imports a sheet and makes it easy to print stickers. Can do various types of computer vision relevant fiducial markers.

## Installation
1. Pull the repo to some folder.
2. Install the requirements.
3. Run avery_5267_gui.py

## Usage
I recommend just setting up a google sheet with the columns ID, Line 1, Line 2, Line 3, Line 4, Line 5, Side Line. Then just formatting it however you want.

Export as csv and the import tab will bring it in easily.

Use the mouse wheel in the preview space to skip used stickers.
